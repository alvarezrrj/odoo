## Migration Script Template

### Complete Migration Script
```python
# migrations/18.0.1.0.0/pre-migrate.py
from openupgradelib import openupgrade

def migrate(cr, version):
    """Pre-migration script for l10n_ar 17.0 → 18.0"""
    
    # Remove deprecated fields
    openupgrade.drop_columns(cr, [
        ('res_partner', 'l10n_ar_special_purchase_document_type_ids'),
        ('account_move', 'l10n_ar_currency_rate'),
    ])
    
    # Clean up relation table  
    cr.execute("DROP TABLE IF EXISTS res_partner_document_type_rel CASCADE")
```

```python
# migrations/18.0.1.0.0/post-migrate.py
from openupgradelib import openupgrade

def migrate(cr, version):
    """Post-migration script for l10n_ar 17.0 → 18.0"""
    
    # Update company external ID references
    company_mappings = [
        ('l10n_ar', 'company_mono', 'base', 'company_mono'),
        ('l10n_ar', 'company_exento', 'base', 'company_exento'),
        ('l10n_ar', 'company_ri', 'base', 'company_ri'),
    ]
    
    for old_module, old_name, new_module, new_name in company_mappings:
        cr.execute("""
            UPDATE ir_model_data 
            SET module = %s, name = %s
            WHERE module = %s AND name = %s
        """, (new_module, new_name, old_module, old_name))
```
